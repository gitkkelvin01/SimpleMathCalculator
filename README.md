# SimpleMathCalculator

## Calculadora Simples que utiliza algumas funcionalidades como:

Operações Matemáticas: Realiza cálculos de soma, subtração, multiplicação e divisão.

Função Undo (Desfazer): Permite apagar o último caractere digitado, evitando ter que limpar toda a conta em caso de erro.

Função Clear (Delete): Limpa todo o visor da calculadora com um único clique.

Suporte a decimais: Permite cálculos com números de ponto flutuante.

Função Space: Inclusão de um botão de espaço para organização visual da expressão matemática durante a digitação.

## Tecnologias Utilizadas:

HTML5: Estruturação semântica dos botões e do display.

CSS3: Estilização utilizando Grid Layout para o alinhamento perfeito das teclas.

JavaScript (Vanilla): Lógica de manipulação do DOM e processamento de cálculos através da função eval().

Como visualizar:
Basta abrir o arquivo index.html em qualquer navegador ou acessar o link da preview no CodeSandbox.
